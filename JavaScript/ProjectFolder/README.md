# BriTay2664 Portfolio Project

## Overview

This portfolio webpage was created by Brian Taylor as a personal web development project. The page includes information about me, featured content, project links, a contact form, dark mode, a welcome modal, tooltips, and JavaScript-powered dynamic project cards.

## Languages and Technologies Used

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Session Storage
- JSON
- GitHub

## Features

- Navigation menu
- About Me section
- Skills list generated with JavaScript
- Dynamic project cards created from JavaScript objects
- Project data stored and retrieved using sessionStorage
- Dark mode saved with localStorage
- Contact form with timed confirmation message
- Welcome modal
- Tooltip on the submit button

## Dependencies

No external libraries are required. The project runs using standard HTML, CSS, and JavaScript.

## How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Make sure `styles.css` is in the same folder as `index.html`.
