Agile Final Project - Menu-Driven C Application

Files included:
1. Application.c
2. Menu.h
3. MainLoop.h
4. writeFile.h
5. readFile.h
6. calculateOne.h
7. calculateTwo.h

Suggested compile/run steps in Visual Studio:
1. Create a new C/C++ console project.
2. Add Application.c as the main source file.
3. Add every .h file to the same project folder as Application.c.
4. Build the project.
5. Run the project.

Menu options:
1. Write text to group_output.txt
2. Read text from group_output.txt
3. Calculate rectangle area
4. Calculate the average of two numbers
5. Exit
